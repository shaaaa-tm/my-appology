# Interactive Apology Page

A soft, heartfelt, interactive apology page with gentle messages, GIFs, and soothing background music. This page is designed to allow the recipient to respond at their own pace.

---

#Features

- Soft, warm design: calming background colors and readable fonts.  
- Interactive buttons:  
- “I’m ready to forgive” → shows a thank-you message with a GIF.  
- “I still need time” → cycles through empathetic messages.  
- Background music: soft, looping music to create a calming atmosphere.  
- Responsive design: works on both desktop and mobile devices.  
- Single-page interaction: no redirects needed, everything works on one page.

---

# How It Works

1. **Initial View**: Shows the apology message with two buttons.  
2. **No / “I still need time”** → displays supportive messages one by one.  
3. **Yes / “I’m ready to forgive”** → hides the apology container and reveals a thank-you message with GIF.  
4. **Background Music**: Plays automatically (may require a tap on mobile devices).
